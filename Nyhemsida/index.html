About

This pen contains the styles needed for an animation which can be used to toggle (for example) a persons description on a personal website or profile.

It would be cool to have this animation play on a scroll event. Or perhaps on click of a specific button. It is currently controlled by a click event on $(document).

The animation is controlled by adding a .collapsed class to the .header element.

Feel free to use this in your project. Try come up with the most creative way to toggle the animation!

If the main box looks a bit off, "it's too wide for the screen" your screen is too narrow to contain a 600px wide box.
